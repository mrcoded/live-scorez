## Branch Naming Convention
Ensure your branch follows these naming conventions:
- **Features**: `feat/(name-of-feature)` (e.g., `feat/database-orm-setup`)
- **Bug Fixes**: `fix/(description)` (e.g., `fix/auth-leak`)
- **Documentation**: `docs/(description)` (e.g., `docs/api-endpoints`)
- **Chore/Refactoring**: `chore/(description)` or `refactor/(description)`
